# 🎬 Netflix Data Analysis

## 🚀 Overview
This project analyzes Netflix dataset to uncover trends in content type, country distribution, ratings, and release patterns using Python.

---

## 🛠️ Tools Used
- Python  
- Pandas  
- Matplotlib  

---

## 📊 Key Insights
- Movies are more than TV Shows on Netflix  
- United States is the top content-producing country  
- Most content is rated TV-MA  

---

## 📈 Advanced Insights
- Content increased rapidly after 2015  
- Most movies are around 90–120 minutes  
- Netflix has expanded globally
